Formatting Help

You can use these tags to format your text!

Tag Effect Example Result
<em>TEXT</em> Make TEXT emphasized That is <em>so</em> true! That is so true!
<strong>TEXT</strong> Make TEXT bold <strong>Phenomenal, cosmic power!</strong> Phenomenal, cosmic power!
<img src="URL"> Displays the image at the URL <img src="/assets/pets/standard_3.png"> <%= image_tag 'pets/standard_3.png' %>
<a href="URL">TEXT</a> Makes TEXT a link to URL Visit <a href="https://www.terrepets.com">TerrePets</a>! Visit TerrePets!
